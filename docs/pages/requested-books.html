Requested Books

This section is for processing requests from registered members. Here librarian can see all book requests and prepare books to issue. There is ability to configure email notifications about receiving new book requests by librarian and email sending to member about changing status of request (request can be accepted or rejected).

Member(reader) will see on this page all own requested books and can create new requests.
